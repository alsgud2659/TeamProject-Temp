안전거래를 URL 로 보내준다.
판매자와 카톡이나 문자를 하다가 보면 상대방이 안전결제 페이지라며 URL 을 보내주는데 이런경우 99.9% 의 확률로 사기입니다.
정상적인 안전결제 페이지 URL 은 네이버의 경우 https://order.pay.naver.com/order... 로 시작합니다.
URL 에 https 가아닌 http 로 시작하는 URL 은 가짜입니다.

급처분!! 지금 거래 가능한분만 찾습니다.
시세보다 저렴하게 올려 사람들을 현혹하고 지금 거래할 수 있는 분한테만 판매한다며 구매심리를 부추기는 방법이다.
이런 판매자의 제품을 구매할 때는 의심 또 의심 해야 한다. 아무리 급해도 하루란 시간이 없을 정도로 급박하면 중고 거래를 하진 않을 테니까요.

판매자가 조심해야하는 사기입니다.
대표적으로 판매자가 조심해야하는 사기인 3자 사기이다.
판매자가 물건을 판매한다고 글을 올리고
사기꾼은 물건을 산다고 하면서 시간을 끌면서 판매자가 올린 물건 판매글을 자신이 올린 글인것처럼 다시 올려
다른 구매자를 찾아 판매자의 계좌에 입금을하게하고 사기꾼이 판매자에가 물건을 받아가는 방법입니다.
물건을 못받은 구매자는 판매자를 사기로 신고를 하게되고 판매자는
돈도 받고 물건도 줬지만 결국 책임은 판매자가 져야합니다.

안전거래는 중고거래에서 안전한 거래방법으로 분류됩니다.
중고거래시 안전한 결제방법으로 분류되는 안전거래 이용시 수수료가 발생하기도 합니다. 하지만 전결제 수수료를 핑계로 해서 구매자에게 재입금을 요구하는 사기 피해가 발생하고 있습니다.
보통 사기꾼은 안전결제 수수료가 빠진 금액이 결제되었으니 수수료를 포함한 금액을 재입금 해달라고 합니다.
이미 지불한 금액은 업체정산 이후 몇일 뒤 통장으로 환불된다.라고 속이면서 말입니다.
